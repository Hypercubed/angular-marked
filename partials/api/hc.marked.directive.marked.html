marked
directive in module hc.marked

Description

Compiles source test into HTML.

Usage

as element:
<marked
       [marked="{expression}"]
       [opts="{expression}"]
       [src="{string}"]>
</marked>
as attribute
<any marked="{expression}"
     [opts="{expression}"]
     [src="{string}"]>
   ...
</any>

Parameters

ParamTypeDetails
marked
(optional)
expression

The source text to be compiled. If blank uses content as the source.

opts
(optional)
expression

Hash of options that override defaults.

src
(optional)
string

Expression evaluating to URL. If the source is a string constant, make sure you wrap it in single quotes, e.g. src="'myPartialTemplate.html'".
